# Garissa Million Trees

Garissa Million Trees is a frontend application that supports the mission of planting and protecting one million trees in Garissa County, Kenya. This site shares the project story, tracks progress, and helps engage donors, volunteers, and partners.

## 🚀 Tech Stack

- **Framework:** React (JSX)
- **Build tool:** Vite
- **Styling:** CSS
- **Linting:** ESLint

## 📁 Project Structure

- `src/`
  - `App.jsx` – root application layout
  - `main.jsx` – bootstraps React into the DOM
  - `components/` – UI components (Hero, Gallery, ImpactStats, etc.)
  - `assets/` – static assets (images, icons)
- `public/` – static files served as-is

## ⚙️ Getting Started

### Prerequisites

- Node.js 18+ (LTS recommended)
- npm (bundled with Node.js) or Yarn

### Install

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open http://localhost:5173 to view the site locally.

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
